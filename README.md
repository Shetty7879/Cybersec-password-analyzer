# Password Strength Analyzer

A Cyberpunk-themed Password Strength Analyzer built with HTML, CSS, and JavaScript.

## Features
- **Real-time Strength Analysis**: Instant feedback with visual meter and text.
- **Cyberpunk Theme**: Neon aesthetic with Light/Dark mode.
- **Strong Password Generator**: One-click secure password suggestion.
- **Mobile Responsive**: Fully optimized for mobile devices.
- **Secure**: All analysis happens locally in your browser.

## How to Run Locally

### Option 1: Double Click (Simplest)
Simply open the folder and double-click `index.html`. It will open in your default browser.

### Option 2: Using the Python Script
1. Open a terminal/command prompt in this folder.
2. Run: `python run_server.py`
3. This will start a local server and automatically open the website.

## How to use on Mobile (GitHub Pages)

To access this on your phone, you can host it for free using GitHub Pages:

1. Go to your repository settings on GitHub: [https://github.com/Shetty7879/Cybersec-password-analyzer/settings](https://github.com/Shetty7879/Cybersec-password-analyzer/settings)
2. Scroll down to the **"Pages"** section (on the left menu).
3. Under **"Source"**, select `Deploy from a branch`.
4. Under **"Branch"**, select `main` and click **"Save"**.
5. Wait a minute, and GitHub will give you a link (e.g., `https://shetty7879.github.io/Cybersec-password-analyzer/`).
6. **Open that link on your phone!**
